To modify your portfolio (e.g., changing text, adding a new project, or updating colors), follow these steps to ensure your changes go live:

* **Edit the code:** Open your project folder in your code editor (like VS Code), make your desired changes in `app/page.tsx`, and save the file.
* **Stage your changes:** Open your terminal in the project folder and run `git add .` to track the updates.
* **Commit the changes:** Run `git commit -m "Describe your update here"` to create a new save point for your modifications.
* **Push to GitHub:** Run `git push origin main` to upload your changes to your GitHub repository.
* **Automatic Deployment:** Vercel will automatically detect the new commit on your GitHub repository and redeploy your site with the updates within a minute.

Once the process finishes, your modifications will be visible at your live link immediately.
